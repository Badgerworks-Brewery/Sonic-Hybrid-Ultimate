# Contributing to Sonic Hybrid Ultimate

We welcome contributions to Sonic Hybrid Ultimate! To ensure a smooth and collaborative process, please follow these guidelines.

## How to Contribute

1.  **Fork the repository:** Start by forking the [Sonic Hybrid Ultimate repository](https://github.com/Badgerworks-Brewery/Sonic-Hybrid-Ultimate) to your GitHub account.
2.  **Clone your fork:** Clone your forked repository to your local machine:

    ```bash
    git clone https://github.com/YOUR_USERNAME/Sonic-Hybrid-Ultimate.git
    cd Sonic-Hybrid-Ultimate
    ```

3.  **Create a new branch:** For each new feature or bug fix, create a new branch:

    ```bash
    git checkout -b feature/your-feature-name
    # or
    git checkout -b bugfix/your-bugfix-name
    ```

4.  **Make your changes:** Implement your changes, adhering to the project's coding style and conventions.

5.  **Test your changes:** Ensure your changes do not introduce new bugs and that all existing functionalities work as expected.

6.  **Commit your changes:** Write clear and concise commit messages. A good commit message describes *what* changed and *why*.

    ```bash
    git commit -m "feat: Add a new feature" # for new features
    git commit -m "fix: Resolve a bug"     # for bug fixes
    ```

7.  **Push to your fork:** Push your local branch to your forked repository on GitHub:

    ```bash
    git push origin feature/your-feature-name
    ```

8.  **Create a Pull Request:** Go to the original Sonic Hybrid Ultimate repository on GitHub and create a new pull request from your forked branch to the `main` branch. Provide a detailed description of your changes.

## Code Style and Conventions

*   **Follow existing style:** Maintain consistency with the existing code style throughout the project.
*   **Clear and concise code:** Write readable and self-documenting code.
*   **Comments:** Add comments where necessary to explain complex logic.

## Reporting Bugs

If you find a bug, please open an issue on the [issue tracker](https://github.com/Badgerworks-Brewery/Sonic-Hybrid-Ultimate/issues) and provide the following information:

*   A clear and descriptive title.
*   Steps to reproduce the bug.
*   Expected behavior.
*   Actual behavior.
*   Screenshots or error messages (if applicable).
*   Your operating system and environment details.

## Feature Requests

If you have a feature idea, please open an issue on the [issue tracker](https://github.com/Badgerworks-Brewery/Sonic-Hybrid-Ultimate/issues) and describe:

*   The feature you'd like to see.
*   Why it would be beneficial to the project.
*   Any potential challenges or considerations.

Thank you for contributing to Sonic Hybrid Ultimate!
